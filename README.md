# zulucoda pics
A simple picture slider

Tested with node version 4.6.0

**To Run:**

**Step 1: Install**
`$ npm install`

**Step 2: Start**
`$ npm start`

**Other:**

**Run All Tests**
`$ npm test`

Stack Used:
- Angular-Webpack workflow
- ES6
- SASS
- GSAP (Animation)
- Bootstrap
- Jasmine (Test)
- Karma (Test Runner)
- Babel (ES6)
